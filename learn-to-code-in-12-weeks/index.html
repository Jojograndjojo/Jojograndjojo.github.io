In case you did not notice I am using ghost as a platform for this blog.

"Why ghost?" You might want to ask, well because of that :

def write_code  
  "I can write code snippets :)"
end

And because when I started to walk the web dev path, it was recommended to me.

The thing I didn't know back then was that, to host a ghost blog free on github, I had to use Node. And back then I didn't really know what Node.js was , or git or most importantly a stack trace.
I just thought the command line hated me, when it was just telling me what I needed to do (communication is key in a command line/dev relationship)

Anyway I came a long way in about 3 months. "How?" You may want to ask. I joined the Makers Academy, Europe's leading Web Developer bootcamp (as their website say(it is sometimes hard to pass the right tone when writing, just to be clear, going there was a great idea for me( putting parentheses within parentheses is bad practice)).

Where I went from "Ouais!! I can get my laptop to tell me my own age!" (I happen to be French, Ouais !! means yeah!!). To be able to write applications like the following:

Time for an Adventure

Working in groups, using test driven development , trying to follow agile and stick to the xp values. I will get to talk more about all of this in future posts.
But right now all I am trying to say is sorry to my numerous readers (Haha...), I stopped to publish posts as I was busy getting the skills allowing to... well publish posts but also getting to become a better developer. I hope you will find the strength to forgive me.
